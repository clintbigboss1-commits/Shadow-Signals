# Operative easter-egg images

Drop the spy-operative PNGs (transparent background preferred) in this folder.
Each page picks up its image automatically — no code change needed:

| File           | Page          | Suggested operative                          |
|----------------|---------------|----------------------------------------------|
| `landing.png`  | Landing hero  | Binoculars operative peeking at the phone mockup |
| `markets.png`  | Markets feed  | Operative lurking in the corner, watching bet cards |
| `ghost.png`    | GHOST page    | Operative peering from the shadows           |
| `pricing.png`  | Pricing page  | Operative "spying" on the tiers              |
| `wins.png`     | Wins page     | Operative observing the winning bets         |

Until a file exists, the page simply renders without it (the component hides
itself on a 404). Recommended width: 400–600px source, displayed at 140–180px.
